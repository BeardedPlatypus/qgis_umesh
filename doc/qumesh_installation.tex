\section{Installation}

Copy the \qumesh-dll to the \Cpp plugin directory of QGIS. 
In my development installation it is:\newline
\dir{c:\textbackslash OSGeo4W64\textbackslash apps\textbackslash qgis\textbackslash plugins\textbackslash qgis\_umesh.dll}

The \qumesh.dll need the following dll's, so they should be in the \%PATH\%-environment variable:
\begin{itemize}
    \item qt5core.dll
    \item qt5gui.dll
    \item qt5widgets.dll
\end{itemize}
Next dll's are probably already placed in the \%PATH\%-environment variable when installing QGIS:
\begin{itemize} 
    \item qgis\_core.dll
    \item qgis\_gui.dll
\end{itemize}
